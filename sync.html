Study Time Sync Bridge (deprecated)

This bridge has been retired. Study-time aggregation now goes through a Vercel KV-backed HTTP API at https://shadowing-app.vercel.app/api/study-time, identified by an anonymous UUID stored in each device's localStorage. See cloud-sync-client.js on this same domain for the new client. This page is kept only for backward compatibility with already-cached parent pages.

Historically: this page exposed a postMessage bridge so that sibling apps (shadowing-app, pronunciation-coach) could share daily totals through this origin's localStorage. Now it is not actively used.
